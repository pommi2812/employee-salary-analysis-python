# Python Employee Salary Analysis

This project analyzes an employee salary dataset using Python to extract insights and visualize trends.

## Tools Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Dataset
The dataset contains the following columns:

- ID
- Experience_Years
- Age
- Gender
- Salary

## Analysis Performed

✔ Data loading and exploration  
✔ Summary statistics using Pandas  
✔ Salary analysis by gender  
✔ Experience vs Salary relationship  
✔ Data visualization using charts

## Key Insights

- Average salary of employees
- Highest and lowest salary
- Salary distribution
- Relationship between experience and salary

## Visualizations

The project includes:

- Bar chart: Average Salary by Gender  
- Scatter plot: Experience vs Salary  
- Histogram: Salary Distribution
